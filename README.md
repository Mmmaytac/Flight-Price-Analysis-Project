# Flight-Price-Analysis-Project
Flight Price Analysis Project

##  Project Overview

This project focuses on analyzing airline ticket prices based on multiple features such as airline, source and destination cities, departure and arrival times, number of stops, travel class, duration, and days left before departure.

The objective of this analysis is to explore how different factors influence flight prices and to identify meaningful patterns within the dataset.

##  Dataset Description

The dataset contains the following features:

airline – Airline company

flight – Flight code

source_city – Departure city

departure_time – Time of departure

stops – Number of stops

arrival_time – Time of arrival

destination_city – Destination city

class – Travel class (Economy/Business)

duration – Flight duration

days_left – Number of days left before departure

price – Ticket price (Target variable)
